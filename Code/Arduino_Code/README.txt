CURRENT SKETCH: sketch_BATL_jan01-2021.ino

TO LOG DATA:
1. open putty.exe (located in folder ExtraPuTTY_githubcopy)
2. change session settings: serial line, baud speed
3. change logging settings: browse to correct location (wherever data should be stored)
3. save these settings as default (session settings)
4. hit "Open" to begin logging
5. close session to save data
